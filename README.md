# sistema-gestao-escolar-db - Banco de dados
Projeto acadêmico desenvolvido durante a graduação em Engenharia de Software.

## Descrição
Modelagem de banco de dados para um sistema de gestão escolar, contemplando alunos, professores, turmas, disciplinas e matrículas.

## Conteúdo do repositório
- Diagrama Entidade-Relacionamento (DER)
- Modelo lógico
- Script SQL para criação das tabelas

## Tecnologias e conceitos
- Modelagem de banco de dados
- Diagrama ER
- SQL (nível acadêmico)
